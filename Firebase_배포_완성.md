## 🔧 관리 및 업데이트

### 설치파일 업데이트 방법
1. **새 버전 빌드** → `python build_installer.py`
2. **Firebase Console** → Storage → 파일 교체
3. **토큰 변경시** → HTML 링크 업데이트
4. **배포** → `firebase deploy --only hosting`
