PC 활동 로그를 분석하여 당신의 생산성과 행동 패턴을 시각화합니다. AI 디지털 행동 심리학자가 분석하는 당신만의 페르소나를 확인해보세요.

수집기 다운로드

3단계로 끝나는 스마트한 관리

1. 수집기 실행

전용 프로그램을 실행하여 평소처럼 PC를 사용하세요. 활동 내역이 안전하게 로컬에 기록됩니다.

2. 로그 업로드

생성된 `raw_activity_log.csv` 파일을 본 사이트에 선택하여 분석 엔진에 입력하세요.

3. AI 분석 리포트

시각화된 통계와 AI가 생성한 당신만의 페르소나 리포트를 즉시 확인하세요.

Security Notice

설치 프로그램 실행 시 Windows Defender 경고가 뜰 수 있으나, 이는 서명되지 않은 실행 파일에 대한 일반적인 경고이며 안전합니다. "추가 정보 → 실행"을 클릭하여 계속 진행해주세요.
