### 🔧 **고급 매칭 시스템**
```javascript
// 지원하는 매칭 규칙들:
- keywords: 모든 키워드 포함 확인
- exact_match: 정확히 일치하는 문자열
- exclude: 제외할 키워드들
- any_of: 하나라도 포함되면 매칭
- priority: 우선순위 (높을수록 먼저 확인)
```

## 🚀 **핵심 개선사항**

### 1. **정확한 분류**
```javascript
// 기존: 단순 문자열 포함 검사
if (activity.includes('YouTube')) { ... }

// 개선: 우선순위 기반 다중 규칙 매칭
sortedCategories.forEach(([id, info]) => {
    if (matchesCategory(activity, activityLower, info)) {
        return info.name;
    }
});
```

### 2. **Chrome 세분화**
- 기존: `Chrome` 하나로 통합
- 개선: **9개 세부 카테고리**로 분리
  - Gmail, Drive, GitHub, Discord, Notion 등

### 3. **충돌 방지**
```javascript
// exclude 키워드로 오분류 방지
"chrome_google": {
    "keywords": ["chrome", "google"],
    "exclude": ["youtube", "gmail", "drive"]  // 제외 규칙
}
```

### 4. **복합 조건**
```javascript
// any_of: Google Workspace 통합
"chrome_workspace": {
    "keywords": ["chrome"],
    "any_of": ["docs", "sheets", "slides"]  // 하나라도 포함
}
```
